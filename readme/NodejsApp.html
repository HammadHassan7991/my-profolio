Features Added in this Project

This app is build using NODEJS with ExpressJs Framework.

  • Each user can add, delete or edit his product. A user can perform these to only those product which he/she added
  • Each user and product data is stored in MongoDb database
  • Authentication through session and cookies and each session is stored in database
  • Notify user through email while signup and reseting password using third party Nodemailer Service.
  • Authorization to each user so that only valid user can perform admin related tasks
  • Validation in Login Signup Add-Product and Edit-Product Page so that only valid data can be added by user
  • Protection again Cross Site Attacks using CSRF Protection. Csrf tokens are included in each admin related requests like login signup add edit delete add-cart
  • Proper Error Handling through Middlware so that for every critcal error app display 500 error page instead of crashing
  • Add to Cart, Order page & Stripe Payment. Add to card and order data is also stored in database
  • File upload and download. Each user can download INVOICE for related order
  • Async request to server & DOM manupulation for deleting product so that page doesnt reload while deleting product
  • Pagination is also included for displaying all products
